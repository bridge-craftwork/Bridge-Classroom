Built and maintained by Rick Wilson and friends. Everything here is free to use, free to inspect, and free to build on. No ads, no subscriptions, no tracking.

Student tools

bridge-craftwork / bridge-classroom Live
The solo practice app, hand curator, and this website. Vue 3 SPA with a Cloudflare-tunneled backend.
bridge-craftwork / Bridge-Game-Analysis CLI · web coming
Analyze club game results board by board. Fully functional as a CLI today; web interface in development.
bridge-craftwork / pbs-bbo-extension 4 browser stores
Browser extension for BBO. Practice curated bidding scenarios directly on bridgebase.com. Available for Chrome, Firefox, Safari, and Edge.
📚

Lesson collections

bridge-craftwork / baker-bridge Live
The Baker Bridge lesson collection — bidding conventions and play. Includes matching printable lesson plans, handouts, and dealer files for teachers.
bridge-craftwork / bridge-lessons-by-grant-robinson Live
Lesson collection by Grant Robinson. Precision bidding with transfer positive responses to strong 1♣ opening bids.
Collection by Grant Robinson
ADavidBailey / Practice-Bidding-Scenarios Deal library
Pre-bid PBN deal files organized by bidding scenario. Primary input source for the Hand Curator. The bba-filtered/ folder is used directly by the curator workflow.
Maintained by David Bailey
🔧

Teacher & author tools

bridge-craftwork / bridge-wrangler CLI · multi-platform
Process and transform PBN files — filter, sort, split, and reformat bridge deal collections.
bridge-craftwork / pdf-handouts CLI · multi-platform
Package PDF handouts for class distribution — adds headers, footers, and page numbers to teacher materials.
bridge-craftwork / dealer3 CLI · multi-platform
Generate bridge hands to specification. A Rust rewrite of the classic Dealer algorithm, retaining the original scripting interface.
bridge-craftwork / bridge-solver CLI · multi-platform
Double-dummy analysis for bridge deals. A Rust rewrite based on the Bo Haglund double-dummy solver algorithm.
💡 All CLI tools ship with pre-built binaries for macOS, Windows, and Linux. No build toolchain required — download and run. Source code and build instructions are in each repo. dealer3 and bridge-solver are Rust rewrites based on established open-source algorithms.
